GitHub Repositories Overview

Backend Code

The backend is built using ASP.NET Core with a RESTful API architecture. It manages tasks and users with full CRUD functionality, integrates with MongoDB for persistent storage, and uses proper DTOs, services, and controllers for a clean architecture. Pagination, filtering, and enum-based status handling are also implemented.

The frontend is developed with Angular and Angular Material for a clean, responsive UI. It communicates with the backend API for data, uses services for state management, implements pagination and filtering, and includes modals for displaying extra information such as GitHub repository details. Styling follows Material Design guidelines.
